📊 供应商资质对比看板

基于「寻源效率表」模板 · 风鸟数据自动登录抓取 · 多供应商一键对比

v1.0 Windows 10/11 64位 解压即用 无需 Python 内置 Chromium

📥 下载分发包

⬇️ 下载分发包 (v1.0)
文件大小：约 350 MB（zip）
解压后：约 800 MB
含：Python 运行时 + Chromium + 看板程序

🚀 使用步骤

  1. 下载 上方 zip 包（约 350MB），解压到任意位置（建议桌面或 D 盘）
  2. 双击 供应商看板.exe，会自动弹出 cmd 窗口 + 系统默认浏览器
  3. 看板右上角点 「🔐 登录风鸟」，弹出的 Chromium 登录风鸟一次（登录态自动保存）
  4. 回到看板，输入公司全称按 Enter，自动从风鸟抓取 9 项资质
  5. 多家供应商以列对比方式展示，可单元格编辑、一键导出 Excel

⚠️ 注意事项

首次启动：需 5-10 秒加载 Chromium，请耐心等候。
关闭 cmd 窗口：= 停止服务，看板就用不了，需重新双击 exe。
杀软误报：PyInstaller 打包的 .exe 偶尔被误报，加白名单即可。
风鸟登录态：保存在 fengniao_user_data/ 文件夹，删了就需重新登录。
端口：占用 8765，如冲突联系开发者改端口。
